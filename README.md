# ✨family-doctor-system-admin ( 家庭医生管理系统 )

**目前仅是基础骨架**


## 🏆运行程序
```sh
npm i -g yarn # 如果没装过yarn
yarn install # or npm install
yarn dev # or npm run dev
```

## 🎁依赖

vue2全家桶，elementui，mockjs,  axios
